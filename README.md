# 旅游目的地推荐网站

面向学生群体的周末/假期旅游目的地推荐 + OTA导流平台

## 🚀 项目状态

**✅ 已正式上线**

**当前阶段：** 数据观察和优化阶段

**核心目标：** 验证"钱的通路是否通电"

---

## 📊 项目概览

### 核心功能

1. **推荐优先首页**：打开直接看到推荐，无需操作
2. **高转化推荐卡片**：CTA醒目，文案"像钱"
3. **OTA跳转链接**：携程/飞猪/去哪儿返佣
4. **信任信号模块**：学生真实使用信号

### 技术栈

**前端：**
- Next.js 14 + TypeScript
- Tailwind CSS
- 部署：Vercel

**后端：**
- Node.js + Express
- 数据：JSON文件
- 部署：Railway/Render

## 📊 数据观察指标

### 必须监控的3个指标

1. **首页推荐卡CTR**（点击率）- 目标：≥5%
2. **CTA点击次数** - 目标：每天≥10次
3. **是否真实跳到OTA** - 目标：至少1次

### 不要监控

- ❌ 转化率（那是OTA的事）
- ❌ 成交金额（那是OTA的事）
- ❌ 用户停留时间（不是核心指标）
